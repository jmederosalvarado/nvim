# Package Management

Never manually patch package manager lock files or any file that should be autogenerated.

# Worktree and NVIM_APPNAME

Since this config is checked out in a worktree at <check pwd>. In order to launch nvim using this config you need to set the `NVIM_APPNAME` env var properly at launch, i.e. `NVIM_APPNAME=<basename_of_config_folder> nvim`.


## Migration

This worktree is being used to migrate away from lazy.nvim to the builtin nvim package manager introduced in nvim 0.12. We're also using this oportunity to simplify configs and whenever possible migrate to neovim builtin features over plugins.
